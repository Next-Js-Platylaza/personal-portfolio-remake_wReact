## Instructions:

Recreate your personal portfolio as a NextJs application. Have a carousel of links to (or iframes of) hosted projects you have created. Have a section that links to articles you have written or will have written (you can populate them with random lorem ipsum using something like fakerjs if you want). Allow users to create comments on articles, and to view, edit and delete those comments if they have created an account and have logged in. Use NextAuth (AuthJs) with at minimum the Credentials provider (and Google Oauth for a challenge) for authentication. Enable logging out. Access the NWA API to get the most recent forecast for Cache Valley, UT and display it the navbar (for an extra challenge, use JavaScript's Geolocation API to get a forecast for wherever the user is logged in, which will require asking the user if they are comfortable sharing their location). Have a contact form that saves inquiries to a database of your choice.
